Indicating copy 2, 3, 4, etc. of an object in Voyager:

To indicate copy number in Voyager cataloging, do NOT use the copy number on the item record. The item record copy number should always be set at "0" on new records.

Copy 1 of an object is assumed--no special marking is required even if more than one copy is in hand when cataloging.

Record the copy number for any copy beyond the first in the MFHD 852 field in subfield "t". Subfield "t" should appear directly after |b and its contents.

852 _0 |b f |t 2 |h DG225.B79 |i B747 1995

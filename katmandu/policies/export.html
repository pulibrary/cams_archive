Marking / Re-Marking Records for Export in Voyager
The bibliographic record of all records new to the Voyager database which have been fully-cataloged should be marked for export. Records new to the database which are not fully-cataloged (local cataloging, order records, etc.) should NOT be marked for export.

In addition, existing Voyager catalog records which receive significant changes may need to be marked/re-marked for export:

  • All PCC records, whether or not they were created by Princeton, should be re-marked for export when a significant change is made to any part of the record.

  • Other fully-cataloged records: Significant changes made to bib records may be re-marked for export to RLIN at the cataloger's discretion.

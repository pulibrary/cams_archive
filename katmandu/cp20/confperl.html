Locating authors on the Internet

If the author is affiliated with a university, try the university's web site. (You can find university addresses by using a search engine to find the university site or by using a guide site. geowww.uibk.ac.at/univ/ has links to many universities by country.

Use a search engine to search for the author's name. To find John Doe, a hydrographer, a Boolean search for "john doe" and "hydro*" will find an association home page which lists an address for Doe.

Check the publisher's home page. If the URL is not listed in your book, use a search engine to find it, or use one of the directories listed on Yahoo's publishing directory page.

4xx references: Corporate Names: Variant names

Make references from variant forms of the name chosen for the heading. Generally make only one reference from each variant.

    Exception: If the name chosen for the heading is not in English, and there are one or more English forms, also make references from variants of one of the English forms.
For headings entered in direct order, include a reference from the parent body, if appropriate.
Note that this may entail also establishing the parent body (for the reference) in a separate authority record.

Updates to existing NAF records

In this section are listed the most common cases in which an existing NAF record should be updated, and those cases in which existing records cannot be changed.

To initiate a change to an existing NAF record, perform the following steps in Connexion:

  • Call up the authority record (you must logon to the server)
  • Choose "Lock Master Record" from the Action menu
  • Make the changes
  • Save the adjusted record to the online save file
  • Print the record
  • Mark the printout (by circling, highlighting, arrows, etc.) the parts of the record that you changed
  • Initial the printout and put it in the review basket

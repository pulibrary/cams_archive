Producing OCLC authority records

For Authorities Committee members ONLY:

To produce OCLC authority records:

Logon to the naco account

  1. Search each heading and ref. in the authority file before producing

  2. Call up the record by searching the online save file

  3. Make any changes

  4. In the Action menu choose add to authority file to produce new records -- choose replace record to produce updates



To update existing NAF records:

  1. Pull up the existing NAF record
  2. Lock the record by choosing Lock Master Record from the Action menu
  3. Edit the record
  4. Choose Replace Record from the Action menu to produce the record

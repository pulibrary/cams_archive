Withdrawal procedure (for HaRMS staff only)

If no purchase order is attached:

If only copy:
Delete item and holdings record. Before deleting bibliographic record add a 599 field with the following note “Record deleted, date (mm/dd/yy) and initials”. Be sure to follow the wording and format below exactly. Save the bib record and then delete.

599

 

 

|a Record deleted 09/03/06 mwc

If multiple copy:
Delete item and holdings record.

If a purchase order is attached:

If only copy:
Delete item record and suppress the bibliographic record. Add an 866 field in the holdings record with a non-public note following format below

866

 

0

|x Withdrawn.

If multiple copy:
Delete item record and suppress the holdings record. Add an 866 field in the holdings record with a non-public note following format above.

Microfilms physical processing:

  • Discard boxes microfilms are in
  • Remove microfilm from reels and discard
  • Give empty reels to Microforms Services if reels have square holes on both sides.
  • If reels don’t have square holes on both sides, discard them.

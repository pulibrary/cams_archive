Annexing single volume monographs
8/1/00

When in doubt or if there are ANY questions regarding this procedure, call Maryann Langer, Catalog Unit V, 8-2309.

NOTE: Before any annexing work is done, the Annex Notification form should have been submitted and approved.

NOTE: No more than one copy of a monograph or bookset may be located in the annexes. If there is already a copy in either annex, you must either keep the item(s) or send it to Cataloging Unit V for withdrawal or further inspection. (see Revised Annex Policy, Addendum to Library Council Summary, 7/29/1993).

NOTE: Each volume being annexed must have an item record in Voyager. If not, add item record(s) to Annex mfhd as per step 7 in this procedure.

  • Log onto Voyager cataloging module.

  • Call up the record. Examine the bibliographic record to verify title -- what you are sending must match bibliographic information as listed in the Voyager record.

    NOTE: It is helpful to use the "Show associated records" icon to display the bibliographic tree and see the holdings record(s) and item record(s) that are linked to the bibliographic record.

  • Display the holdings record by clicking on the "Strapped books" icon. Update the location in the 852 field. Place the cursor at the beginning of the location, manually retype in a new location and delete the old location, or with the cursor in the |b field press ctrl/L and choose your location from the list. The location must be in lower case. At the end of the call number add a transfer note in a subfield x.
    The F9 key will give you the delimiter symbol.
    Example:
    852 0   |b anxb |h QE350 |i .S44 1984 |x tr fr sg

  • Click on the boat to save the record to the database.

  • Display the item record by clicking on the "Stacked books" icon. Verify that the barcode in the item record matches the item you are annexing. Change the loan code in the item type to Gen if necessary. Change the location in the perm. loc. box.

  • Click on the boat to save the changes of the record to the database.

    NOTE: In accordance with the Revised Annex Policy, Addendum to Its Summary, 7/29/1993, Library Council affirmed that the Annexes are a separate location and not an extension of the holdings of any other branch. Either anxa or anxb must be used as the location of record for any materials transferred to the Annexes.

  • If there is no item record, add it at this point. See Adding Holdings and Item records to Voyager for monographs and booksets .

  • Prepare volume(s) being sent:

    On verso of title page (in pencil), cross out your location and write above it (AnnexA) or (AnnexB) as appropriate.

    Place appropriate Annex sticker on spine. If possible, cover your location designation. If call number label is on cover of book, place Annex sticker next to it.

    Enclose an Annex Packing Slip with each box.

When in doubt or if there are ANY questions regarding this procedure, call Maryann Langer, Catalog Unit V, 8-2309.

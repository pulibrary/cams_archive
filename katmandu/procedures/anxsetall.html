Annexing Serials and Booksets - entire runs
8/1/00

When in doubt or if there are ANY questions regarding this procedure, call Maryann Langer, Catalog Unit V, 8-2309.

NOTE: Before any annexing work is done, the Annex Notification form should have been submitted and approved.

NOTE: Serial volumes may not be duplicated in an annex and a location. If what you want to send to the annexes is also held in another location, you may not annex it. No more than one copy of a bookset may be located in the annexes, (see Revised Annex Policy, Addendum to Library Council Summary, 7/29/1993) If there is already a copy in either annex, you must either keep the item(s) or send them to Cataloging Unit V for withdrawal or further inspection.

NOTE: Each volume being annexed must have an item record in Voyager. If not, add item record(s) to Annex mfhd as per step 7 in this procedure.

  • Log onto Voyager cataloging module.

  • Call up the record. Examine the bibliographic record to verify title -- what you are sending must match bibliographic information as listed in the Voyager record. Be aware of title changes.

    NOTE: It is helpful to use the "Show associated records" icon to display the bibliographic tree and see the holdings record(s) and item record(s) that are linked to the bibliographic record.

  • Display the holdings record by clicking on the "Strapped books" icon.

    To edit a field: Use the usual Windows cut/copy/paste and overwrite capabilities.
    To add a field:
        Use the F3 function key to add a line above the selected line
        Use the F4 function key to add a line below the selected line
    To delete a field, select the field and hit the delete key.
    To insert a delimiter symbol use the F9 function key

  • Update the location in the 852 field. Place the cursor at the beginning of the location, manually retype in a new location and delete the old location, or with the cursor in the |b field press ctrl/L and choose your location from the list. The location must be in lower case. At the end of the call number add a transfer note in a subfield x. The F9 key will give you the delimiter symbol.

    Example:
    852 0   |b anxb |h QE350 |i .S44 1984 |x tr fr sg

  • Verify the summary holdings statement in the 866 |a field.
    If there is an obsolete "current issues in" note delete it.

    If there is any discrepancy between the holdings statement and the volumes actually held, refer to the Adding Holdings and Item Records to Voyager for Serials, section on Adding and displaying a holdings record

    NOTE: The LACKS note is not be used for missing items.

    Example:
    866 0   |a Vol.71, no.1 (spring 1984)-v. 84, no.6 (Dec. 1997)
    866 0 |z LACKS: v.80, no.5
    866 0 |z CURRENT ISSUES IN: (SW)
    866 0 |x DESIGNATOR: vol., no., etc.

    OR

    If you don't wish to update 866 fields yourself, make a printout of the holdings record and indicate on it what the updated holdings are. Send this printout to Catalog Unit IV for updating when you have finished the following steps.

  • Click on the boat to save the changes of the record to the database.

  • Display the item records by clicking on the "Stacked books" icon. Retrieve them one by one or all at once
    Or,
    Retrieve each item record by the barcode by selecting from the bemu bar Record>Retrieve by barcode, and scanning the barcode.

  • Verify that the barcode in the item record matches the item you are annexing. Change the loan code in the item type to Gen if necessary. Change the location in the perm. loc. box to anxa or anxb.

  • Click on the boat to save the record to the database and then close it.

  • Repeat steps 7 to 9 for each item record.

  • When all item records has been changed. Click on the "Show associated records" icon to display the bibliographic tree, with the holdings record(s) and item record(s) and check that the annex location displays for all item records.

    NOTE: In accordance with the Revised Annex Policy, Addendum to Its Summary, 7/29/1993, Library Council affirmed that the Annexes are a separate location and not an extension of the holdings of any other branch. Either anxa or anxb must be used as the location of record for any materials transferred to the Annexes.

  • If there is no item record for a particular volume, add it at this point. See the section on Adding an item record in the Adding Holdings and Item records to Voyager for monographs and booksets procedure or Adding Holdings and Item records to Voyager for serials procedure

  • Prepare volume(s) being sent:

    On verso of title page (in pencil), cross out your location and write above it (AnnexA) or (AnnexB) as appropriate.

    Place appropriate Annex sticker on spine. If possible, cover your location designation. If call number label is on cover of book, place Annex sticker next to it.

    Enclose an Annex Packing Slip with each box.

When in doubt or if there are ANY questions regarding this procedure, call Maryann Langer, Catalog Unit V, 8-2309.

What can be bib linked

Bib linking

Princeton is using Voyager's bib linking feature primarily to create links between records for print and electronic versions of the same title. Profiles have been established to link between e-books and print counterparts, and between e-serials which have separate catalog records and their print counterparts.

Special requests for links between different versions of a title in other formats and for links between related electronic/print resources will be accommodated within reason.

At present, we will not expand bib linking to include traditional relationships already covered by data in the catalog record like earlier/later serial titles, or related print resources like a large set and its index.

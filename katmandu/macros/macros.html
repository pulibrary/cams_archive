Macros

We are currently using Macro Express v. 2.1e. All centrally-written and supported macros are written for this version of Macro Express.

Having a problem with a specific macro or macro file? Consult this troubleshooting guide before reporting the problem.

All Transfer.mex

This file is used for transfers and recapping. Contact Marie Wange Connelly for training in transfer activities. Documentation for using this macro file is available in the cataloging documentation for recap (see the appropriate subsection under Transfer workflow). To obtain this file or report a problem with these macros, contact Don Thornbury or Joyce Bell.

voymacs.mex

This file is used for cataloging and acquisitions module work. It provides macros to input diacritics, add 902s and 904s, process LC cataloging, overlay records, receive orders from bib records, etc. A list of macros/menus of macros in the file can be seen by invoking the ctrl+shift+h macro. Report problems with the order macros to Carol Fagundus; to obtain this file or report other problems, contact Joyce Bell.

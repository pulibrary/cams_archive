.A6-.A99 Table
The cutter table .A6-.A99 is used in the Biography table and tables of the Class P, subclasses P-PZ.

It is broken down as follows:

Aa-Af------ .A6-699
Ag-Al------ .A7-799
Am-Ar----- .A8-899
As-Az----- .A9-999

TheTranslation Table can be applied to this table.

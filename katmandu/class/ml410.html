ML410-ML429

The following procedure should be followed to assign the proper ML410-ML429 number:

  1. Find the composer/performer’s authority record. If there is a class number in the 053 field (e.g., Elgar, Edward = ML410.E41), use it and cutter for main entry. If no 053 is present in the authority record, go to step 2.

  2. Search Voyager for a record about that composer/performer. Use records for books, not sound recordings, scores, etc. Prefer more recent records. If you find a record, use the class number and cutter for the main entry. If this search yields no hits, go to step 3.

  3. Assign a number to the composer/performer, fitting him/her into our collection using a subject search in our catalog .

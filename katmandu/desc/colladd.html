Adding to an Existing Collection
Whenever possible, prefer creating a new collection to adding to an existing collection. A second set of similar material can be cataloged under the same title with the addition of a number. (e.g., [Modern Arabic poetry, Egypt, 2]).

When it is necessary to add to an existing collection, follow these instructions:

Boxed collections

Adjust the table of contents for the affected box(es). Put call number labels in each new item (within reason). Adjust the bibliographic record as necessary (date range, size range, 505).

Bound collections

Follow the same procedures as for a bookset.

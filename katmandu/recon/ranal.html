Recons of Analytics

If the call number includes a designator and number, such as

    14094.02.69 no. 55
consider this item to be part of an analyzed set.
  • Trace the title of the set in a 440 field, using the title which appears in parentheses on the card.
  • When pulling cardsets from the Card Catalog, leave the card representing the main set.
  • If the number of the analytic is on its own card, delete it.
  • If it is on the same card with other analytics in the same set, refer it to the Head of DBIU.

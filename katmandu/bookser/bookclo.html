Closings
When adding to a bookset, try to determine if the bookset is closing with the volume that you are adding.
If it is closing with the volume that you are adding, or if you receive a notice from the Order Division that the bookset has ceased, verify/change the following elements of the record:

Fixed Fields (008)

  • Change the 008 Date 1 if necessary.
  • Complete the 008 Date 2 if necessary.
Variable Fields
  • Close the imprint date in the 260 field (when closing without the last piece, the imprint date of the last volume is often present in the "published in" note on the holdings screen.
  • Close the 300 field, if possible.
  • Change the completeness indicator in the 505 field.
Holdings Screen
  • Change the summary holdings note to reflect Princeton's holdings.

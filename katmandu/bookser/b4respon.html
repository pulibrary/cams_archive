Responsibility and entry
Adjust the description and/or notes (245 ‡c, 500, etc.) as necessary. Also, adjust the entry as appropriate to reflect the new description or responsibility, and make appropriate added entries, e.g.,,

Volume 1 (basis for original catalog record) t.p.: by R. Jones and J. Smith

100 Jones, R.
245 Title /‡cby R. Jones and J. Smith
700 Smith, J.

Added volumes 2-6: by J. Smith.

It is clear that J. Smith has assumed predominant responsibility for the bookset. The main entry should be changed to Smith, J. and Jones, R. should be changed to an added entry.

If a bookset is entered under an author heading but subsequent volumes indicate that more than three primary authors are involved, and it is not clear whether any of the authors has primary responsibility, the main entry should be changed to title and the original main entry should be changed to an added entry.

If in doubt whether or not to change the main entry of a bookset please consult the Team Leader.

NOTE: Changes in the main entry are confined to the bibliographic description. The original call number cutter is retained. Book labels are not adjusted.

<titleInfo>

We recommend following AACR2 as the content standard for titles. The primary title will not include a type attribute, all other titles given will include this attribute.

"Translated" titles include both translations provided by the cataloging agency, and translated titles found on the resource.

If the resource lacks a primary title, a displayLabel should be used (i.e. Title devised from content, or Title from ...)

Titles which would appear in notes in a MARC record will remain notes. They will not be given title access.

Core elements for all library metadata records, regardless of schema

At least one record for a project will contain these core elements as a minimum. Wherever feasible, all records for a project will contain these elements.

Title*
   Transcribed title is mandatory if available, if not, supplied title is mandatory

Date*
   Date of creation of original material; archivally this may be date range or bulk dates

Author/Creator*
   An entry for the person(s) bearing primary responsibility for the resource is mandatory if applicable

Publication/Distribution/Origin*
   Recording original item characteristics are mandatory if applicable

Extent*
   Extent information is mandatory "if sensible"

Unique code or Identifier*
   The unique code or identifier is mandatory and may include call numbers, accession numbers, reference codes or a combination unique to our collection

Location*
   Mandatory. Can refer to URL or to location of non-digital originals

Conditions governing access*
   Conditions governing access to original material are mandatory

Language/scripts of resource*
   Mandatory if applicable

Abstract*
   Abstracts are mandatory when applicable (they are applicable to all collection-level records)
